# Синтаксис MarkDоwn

## Выделение текста

Текст можно выделить курсивом, если его обрамить звездочкой (*). Например *вот так\*

Текст можно выделить полужирным, если его обрамить двумя звездочками(**). Например : **текст теперь полужирный\*\*
